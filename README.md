# kotlin-web-api-maven-spring-actuator-simple

## Description
A POC for actuator in springframework.

## Tech stack
- kotlin
- maven
  - actuator

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available at:
- http://localhost
- http://localhost/health
- http://localhost/info

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
